## Using

- Nodejs
- Sequelize ORM
- JWT
- MySQL
- Express
- Nodemon (development)
- PM2 (deploy)
- CORS
- Javascript
- DotENV

## Initial settings

- Install dependencies:

- NPM:

```sh
npm install
```

- YARN:

```sh
yarn install
```

- To run:

```sh
yarn install
```
